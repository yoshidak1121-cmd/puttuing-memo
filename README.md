# パッティングメモ (Putting Memo)

ゴルフのラウンド中におけるパッティングおよびアプローチの記録・管理を行うiPhoneアプリです。

## 概要

プレイヤーが自身のプレーを振り返り、データ分析できる仕組みを提供します。パッティングとアプローチの記録を一元管理し、統計グラフによるパフォーマンス可視化機能を備えています。

## 機能一覧

### 1. パッティング記録
- ボールからカップまでの距離（初期距離）
- ラインの特徴：ストレート、スライスライン、フックライン、わずかなスライス/フック
- 上り/下りの勾配（上り・下り・フラット・横傾斜）
- 成果：成功/失敗
- 結果詳細：残り距離、オーバー/ショートの状態

### 2. アプローチ記録
- 残り距離
- 使用クラブ（PW、AW、SW、LW、各番手アイアン）
- ライの状況（フェアウェイ、ラフ、バンカー、グリーン周り、傾斜地）
- 手前障害物の有無と種類（バンカー、池、木など）
- 結果：次のパッティング距離、上り/下り、距離のズレ

### 3. 履歴管理
- パッティング・アプローチ記録の一覧表示
- 検索・フィルタ機能（ホール番号、成功率、クラブ、ライ状況）
- 時系列での記録確認

### 4. 統計・分析
- 総合成功率、平均距離などのKPI表示
- 距離別成功率グラフ（iOS Charts フレームワーク使用）
- ラインタイプ別成功率
- クラブ別使用率

### 5. データエクスポート
- CSV形式でのエクスポート機能
- ShareSheetを使った外部アプリへの共有

## 環境構築手順

### 必要条件
- macOS 13.0 (Ventura) 以上
- Xcode 15.0 以上
- iOS 16.0 以上のデバイスまたはシミュレーター

### セットアップ

1. リポジトリをクローン
   ```bash
   git clone https://github.com/yoshidak1121-cmd/puttuing-memo.git
   cd puttuing-memo
   ```

2. Xcode でプロジェクトを開く
   ```bash
   open PuttingMemo/PuttingMemo.xcodeproj
   ```

3. ターゲットデバイスを選択してビルド・実行
   - シミュレーター：Product → Run（⌘R）
   - 実機：証明書設定後にビルド

### 依存ライブラリ
現在、外部ライブラリへの依存はありません。グラフ機能には iOS 16 標準の **Swift Charts** を使用しています。

## データ記録チュートリアル

### パッティングを記録する
1. 「パッティング」タブを選択
2. 右上の「＋」ボタンをタップ
3. 以下の項目を入力：
   - **初期距離**：ボールからカップまでの距離（メートル）
   - **ラインの特徴**：ホイールピッカーから選択
   - **勾配**：セグメントコントロールで選択
   - **結果**：成功/失敗を選択（失敗時は残り距離・状態を追加入力）
4. 「保存」をタップ

### アプローチを記録する
1. 「アプローチ」タブを選択
2. 右上の「＋」ボタンをタップ
3. 以下の項目を入力：
   - **残り距離**：グリーンまでの距離（メートル）
   - **使用クラブ**：ホイールピッカーから選択
   - **ライの状況**：セグメントコントロールで選択
   - **手前障害物**：セグメントコントロールで選択
   - **結果**：次のパッティング距離・勾配・距離のズレを入力
4. 「保存」をタップ

### データをエクスポートする
1. 「履歴」タブを選択
2. 右上の共有ボタン（↑）をタップ
3. ShareSheet から「ファイルに保存」または任意のアプリを選択
